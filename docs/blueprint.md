# **App Name**: Thorat Motors India - Modern Industrial

## Core Features:

- Product Catalog: Display trailer and cabin products with specifications, images, and descriptions in a grid layout.
- Mega Menu Navigation: Implement a comprehensive mega menu for easy navigation across all product categories, services, and company information.
- Contact Form: Multi-step contact form for users to request quotes and provide detailed inquiries.
- Responsive Design: Ensure the website is fully responsive and optimized for various screen sizes and devices.
- Office Locator: Display office locations with maps and contact details for multiple cities.
- Dynamic Content Loading: Enable smooth transitions and loading of content across the website, to make it feel snappy and reactive.

## Style Guidelines:

- Primary color: Steel Grey (#3E4850) for text and UI elements to maintain an industrial yet modern feel.
- Background color: Arctic White (#FFFFFF) for a clean and bright aesthetic.
- Accent color: Neon Orange (#FF6B35) for call-to-action buttons, providing a vibrant contrast.
- Primary font: 'Inter' (sans-serif) for headlines and body text, delivering a clean and modern industrial look. Note: currently only Google Fonts are supported.
- Secondary font: 'Roboto Flex' (sans-serif) for body text and technical data, ensuring readability. Note: currently only Google Fonts are supported.
- Clean grid and card-based layout with ample whitespace, thin dividers, and soft card shadows for a Scandinavian industrial design.
- Minimalistic icons to represent product categories and services, enhancing visual appeal without clutter.
- Subtle animations and transitions to enhance user experience, such as hover effects on cards and smooth scrolling.
